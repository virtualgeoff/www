The Client-Side Include...

Server-side includes are incredibly useful, but one thing I've wanted for a while is a client-side include. Sure, you can use iframes and html objects and stuff.

Advantages

  • smaller source code
  • less load on web server
  • good for handheld browsing - the includes are not downloaded but hidden - they are not downloaded in the first place.

Advantages

  • requires javascript
  • doesn't work for browsers that understand javascript but not handheld stylesheets
