.vrml

These are some old VRML demos from 1997–2001.

For a long time they were unusable — they required browser plugins that were no longer available. I've since discovered a JavaScript library called X_ITE that provides a WebGL viewer. No plugin needed.

To get these working again I've removed the old plugin code from the HTML, but made minimal changes to the VRML.
